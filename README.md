# Image-Freehand

* 对于不同的图片可以通过修改深度值depth，光照的俯视角el和方位角az以获得最好的效果
* 代码的具体分析在到我的个人博客 **[RANDOM]<http://www.liangtianming.com>**
* 目前对于人像的转换效果并不好，如眼睛处梯度较小而呈现出变成白色，红色嘴唇边缘梯度较大造成黑色线条较粗，使得嘴唇显厚等，以后有时间会更新算法。
